JMET Documentation Center

The JMET team is part of ETS and works on platform security initiatives for Apple hardware and software (iOS, OSX). Discover the current JMET portfolio and how to contact us.

Activation

  • Falcon
    Crypto backend for carrier activations.
  • Raptor
    Device activation service for internal device verification.

Provisioning

  • Bodhi / Gaya / Maha
    A system that certifies the identity of each SoC that is manufactured and HDCP fusing system for SOCs.
  • Gecko
    System that enables provisioning of IMEI/MEID/MAC addresses. Supports older chip manufacturers and is now in support only.
  • MerlionNew
    System that provides manufacturing pack-out feed.
  • MuhanNew
    System that vends out serial number for Apple devices & device components.
  • TreeckoNew
    The de facto system Apple devices use to provision IMEI/MEID/MAC addresses for all devices with new chip.

Authorization

  • Club-H
    Secure enclave for storing keys for iCloud Keychain.
  • Kirin
    Provides device authorization to support internal repairs.
  • Marina
    Validates OS version of shipped devices to prevent shipping of devices with dev-fused version.
  • PallasNew
    Communicates with devices when new iOS upgrade is available.
  • Tatsu
    Authorizes and personalizes software and firmware installs to make sure that each device is installing authentic and authorized software.

Secured Cloud Storage

  • FDR
    A manufacturing cloud that securely collects and transports device information. This allows Apple to enable in-field recovery and repair, lower cost of manufacturing, and fraud detection.

Security

  • Alder
    Unlocks the SE to enable provisioning of e-sim.
  • Aloha
    Performs a cryptographic pairing between SEP and SE.
  • Moana
    A generic ingest server which may interface with 3rd party vendors for consuming data from them.
  • SeaCookie
    JMET Pairing server for Horizon ASP initiatives from AppleCare.
  • Tabasco
    Provides secure storage of keys and algorithms for iTunes DRM.

Attestation

  • Humbug
    Provides attestation services that validate device authenticity.
